# Text Editor (Electron)

Простий текстовий редактор на **Electron** для перегляду та редагування текстових файлів.

## Структура проєкту

- `main.js` – головний процес Electron, створення вікна та меню
- `preload.js` – безпечний міст між головним і renderer-процесом
- `renderer.js` – логіка роботи з текстовим полем
- `index.html` – інтерфейс користувача
- `style.css` – стилі редактора
- `package.json` – залежності та скрипти

## Функціонал

- Новий файл
- Відкрити файл (.txt)
- Зберегти файл
- Закриття програми

## Встановлення та запуск

```bash
npm install
npm start
```
