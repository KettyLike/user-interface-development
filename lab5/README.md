# React To-Do List

## Опис

Простий веб-застосунок на **React** для ведення списку завдань.
Дозволяє додавати завдання, відмічати виконані та бачити кількість невиконаних.

---

## Можливості

- Додавання нових завдань
- Відмітка завдань як виконаних
- Підрахунок невиконаних завдань

---

## Запуск локально

1. Клонувати репозиторій або завантажити проект.
2. Встановити залежності:

```bash
npm install
```

3. Запустити проект:

```bash
npm start
```

4. Відкрити у браузері: [http://localhost:3000](http://localhost:3000)
