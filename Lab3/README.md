# Текстовий редактор (Qt Framework)

Простий настільний застосунок для перегляду та редагування текстових файлів, реалізований за допомогою **Qt Framework** та мови програмування **Python**.

## Опис

Застосунок реалізує базові можливості текстового редактора. Користувач може створювати нові текстові файли, відкривати існуючі, редагувати їхній вміст та зберігати зміни. Інтерфейс програми побудований з використанням бібліотеки **PySide6**, яка є Python-реалізацією Qt Framework.

## Функціональність

Застосунок підтримує такі операції:

- створення нового файлу
- відкриття існуючого текстового файлу
- редагування тексту
- збереження змін у файл
- завершення роботи програми

Усі дії виконуються через головне меню застосунку.

## Використані технології

- **Python**
- **Qt Framework**
- **PySide6**

## Структура проєкту

```
qt-text-editor
├── main.py
├── requirements.txt
├── views
│   └── main_window.py
└── core
    └── file_manager.py
```

## Встановлення

Встановити необхідні залежності:

```
pip install -r requirements.txt
```

## Запуск програми

Для запуску застосунку необхідно виконати команду:

```
python main.py
```

Після запуску відкриється головне вікно текстового редактора.
