# To-Do SPA

## Опис

Простий веб-застосунок у вигляді **Single Page Application (SPA)** для ведення списку завдань.  
Застосунок дозволяє додавати завдання, відмічати виконані та видаляти їх без перезавантаження сторінки.

---

## Можливості

- Додавання нових завдань
- Відмітка завдань як виконаних
- Видалення завдань
- Миттєве оновлення інтерфейсу без перезавантаження сторінки

---

## Запуск локально

1. Клонувати репозиторій або завантажити проект.

2. Відкрити файл `index.html` у браузері.

або

3. Запустити проект через **Live Server** у Visual Studio Code.
