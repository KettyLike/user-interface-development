# Lab1 – Текстовий редактор (WinForms / WPF)
## Опис проєкту

У межах лабораторної роботи реалізовано два настільних застосунки для роботи з текстовими файлами: один із використанням Windows Forms, інший — із застосуванням Windows Presentation Foundation. Обидві версії забезпечують базову функціональність редагування тексту з можливістю створення нового файлу, відкриття існуючого, збереження змін та завершення роботи програми.

Проєкт розроблено мовою C# на платформі .NET Framework у середовищі Microsoft Visual Studio.

## Принцип роботи

Після запуску застосунок ініціалізує головне вікно, яке містить меню та багаторядкове текстове поле. Користувач взаємодіє з програмою через пункти меню. Обробка дій реалізована за допомогою механізму подій (Click).

При відкритті файлу використовується стандартне діалогове вікно вибору файлу, після чого вміст зчитується методом File.ReadAllText() і відображається у TextBox. Під час збереження застосовується File.WriteAllText(), що записує поточний текст у вибраний файл. Завершення роботи здійснюється через відповідну команду меню.

## Реалізація

Інтерфейс у версії WinForms створюється переважно візуальним конструктором форми. У версії WPF структура вікна описується мовою XAML, а логіка обробки подій реалізується у файлі MainWindow.xaml.cs. В обох випадках застосовується простір імен System.IO для роботи з файловою системою.

## Результат

У результаті розробки створено функціональний настільний текстовий редактор з базовими можливостями роботи з файлами. Проєкт демонструє принципи подієво-орієнтованого програмування, побудову графічного інтерфейсу та взаємодію застосунку з файловою системою.
